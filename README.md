# ETI Marketing Website

Marketing site for **Emerging Technologies, Inc. (ETI)** — Executive IT and AI Advisory.

- **Stack:** Next.js 15 (App Router), TypeScript, Tailwind CSS
- **Content:** JSON + MDX (blog in `content/`)
- **Deploy:** Vercel-ready

## Local run

```bash
# Install dependencies
npm install

# Development
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build and production

```bash
# Build (generates sitemap via next-sitemap)
npm run build

# Start production server
npm start
```

## Deploy (Vercel)

1. Connect the repo to Vercel.
2. Set **SITE_URL** (or `NEXT_PUBLIC_SITE_URL`) to your production URL (e.g. `https://www.emergingti.com`) for sitemap and OG URLs.
3. Deploy. No other env vars required for the MVP.

## Project structure

- `app/` — App Router pages and layout
- `components/` — Header, Footer, Button, Section, ContactForm
- `content/` — `home.json`, `blog/*.mdx`
- `lib/` — `nav.ts`, `site.ts`, `blog.ts`
- `docs/` — `content-map.md`, `design-system.md`
- `public/` — `logo.png`, `robots.txt` (sitemap/robots generated on build)

## SEO and accessibility

- Metadata (title, description, Open Graph, Twitter) per page via Next.js Metadata API
- `sitemap.xml` and `robots.txt` generated by `next-sitemap` on build
- Semantic HTML, focus states, and `prefers-reduced-motion` support

## Logo and contact

- Logo: place ETI logo at `public/logo.png` (PNG or SVG). Current build uses the provided ETI logo.
- Contact form: form UI only in MVP; wire to your API or form service (e.g. Formspree, Vercel server action) for production.
